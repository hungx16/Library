I. 3NF Library Management SystemA professional web application for managing library operations, focusing on database integrity and performance.

II. Overview
This project is a comprehensive Library Management System designed to handle book inventories, reader registrations, and borrowing workflows. The core highlight is the **3rd Normal Form (3NF)** database design, ensuring zero data redundancy and high integrity.
III. Tech Stack

- **Backend:** ASP.NET Core MVC, Entity Framework Core
- **Database:** SQL Server (Structured with 3NF Standards)
- **Frontend:** Bootstrap 5, jQuery, HTML5/CSS3
- **Tools:** Visual Studio 2022, Git

IV. Key Features

- Database Excellence: Fully normalized tables (3NF) to optimize storage and query speed.
- Book Management: Advanced CRUD operations for books, authors, and categories.
- Reader Portal: Manage member profiles, borrowing history, and late return fines.
- Search & Filter: Optimized SQL queries and stored procedures for fast data retrieval.

V. Project Structure

- 3NF_Library.Web: Presentation layer (MVC).
- 3NF_Library.Data: Database context and migrations.
- 3NF_Library.Models: Domain entities and ViewModels.
